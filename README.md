# Smart Engineering Quiz Platform

An interactive quiz website designed for B.Tech students to practice core engineering subjects.

## Features
- Subject-wise quiz (C, C++, Java, Python, OOPS, DBMS, OS, CN)
- Random question generation
- Instant correct answer feedback
- Final score evaluation
- Learning recommendations based on performance

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Select subject and start practicing

## Future Enhancements
- Difficulty-based question pools
- Backend integration
